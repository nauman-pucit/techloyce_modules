Odoo Shopify

Use Odoo Shopify to import / export data between shopify and odoo

Overview

Odoo is a fully integrated suite of business modules that encompass the traditional ERP functionality. Odoo Shopify provides functionality to import orders and customers from your Shopify shop. It also provides functionality to export products from odoo to your Shopify shop.

Key Features And Functions

  • Import orders from Shopify to odoo
  • Import customers from Shopify to odoo
  • Export products from odoo to Shopify

How it Works?

  • Install Python package ShopifyAPI
  • To begin we need to install sales app from odoo and your shop credentials from Shopify
  • you have to privde shop name, api_key, password, secret key
  • Thats it, now you enter into odoo shopify application
  • Here you can select any item from dropdown and press import/export buttons respectively
  • In odoo sales app go to sales->products in mainmenu
  • select any products you want to export to your Shopify shop
  • In action select "Export To Shopify". Products will be exported
